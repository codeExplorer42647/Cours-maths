Chapitre 5.1 — Introduction : Logarithme et exponentielle

1. 🧭 Contexte historique et intuitif

🌍 Origine du mot

Le mot logarithme vient du grec :

  • logos = relation
  • arithmos = nombre→ « relation entre nombres »

📜 Histoire

  • Babyloniens (~2000 av. J.-C.) : utilisaient déjà des tables liant produits et sommes.
  • Napier (1614) : crée les tables logarithmiques pour simplifier les multiplications.
  • Saint-Vincent (1647) : calcule l'aire sous la courbe 1t\frac{1}{t}t1​﻿.
  • Huygens (1661) : identifie cette aire comme un logarithme naturel.
🧠 Intuition :Le logarithme transforme un produit en somme → très utile pour simplifier les calculs avant l'ère numérique.

2. 🧮 Motivation mathématique

📌 Objectif des logarithmes et exponentielles

  • Créer une fonction qui transforme les multiplications en additions.
  • Construire la fonction réciproque d'une exponentielle.
  • Étendre la notion de puissance à tout réel (et pas seulement entier ou rationnel).

3. 🎨 Interprétation géométrique

Soit la fonction f(t)=1tf(t) = \frac{1}{t}f(t)=t1​﻿.

➡️ L'aire sous la courbe entre t=1t = 1t=1﻿ et t=xt = xt=x﻿ :

ln⁡(x)=∫1x1t dt(pour x>0)\boxed{\ln(x) = \int_1^x \frac{1}{t}\,dt} \quad \text{(pour } x > 0)ln(x)=∫1x​t1​dt​(pour x>0)﻿

  • Si x>1x > 1x>1﻿, aire positive
  • Si x<1x < 1x<1﻿, aire négative

Cette aire algébrique donne la définition rigoureuse du logarithme naturel.

4. 🔁 Fonction réciproque : l'exponentielle

  • La fonction ln⁡(x)\ln(x)ln(x)﻿ est bijective de R>0\mathbb{R}_{>0}R>0​﻿ vers R\mathbb{R}R﻿
  • Elle admet une réciproque : la fonction exponentielle

exp⁡(x)=l’unique y>0 tel que ln⁡(y)=x(on note aussi ex)\boxed{\exp(x) = \text{l'unique } y > 0 \text{ tel que } \ln(y) = x} \quad \text{(on note aussi } e^x)exp(x)=l’unique y>0 tel que ln(y)=x​(on note aussi ex)﻿

5. 🧠 Résumé visuel des objectifs

🔢Ce que fait le logarithme naturel ln⁡(x)\ln(x)ln(x)﻿
🧮Transforme un produitensomme
➗Transforme un quotient en différence
🔼Étend la notion de puissance réelle
📐Donne une aire sous la courbe f(t)=1tf(t) = \frac{1}{t}f(t)=t1​﻿
🔁A une réciproque : l'exponentielle exp⁡(x)=ex\exp(x) = e^xexp(x)=ex﻿

6. 📌 Notation clé : le nombre e

  • Il existe un unique nombre réel eee﻿ tel que :

ln⁡(e)=1⇒e=aire sous la courbe 1t entre t=1 et t=e\ln(e) = 1 \Rightarrow e = \text{aire sous la courbe } \frac{1}{t} \text{ entre } t = 1 \text{ et } t = eln(e)=1⇒e=aire sous la courbe t1​ entre t=1 et t=e﻿

Ce nombre e≈2.718e \approx 2.718e≈2.718﻿ est irrationnel, fondamental en mathématiques, analyse, croissance exponentielle, probabilité…

8. 🎯 Ce qu'il faut retenir

  • Le logarithme naturel est une fonction fondamentale définie par une aire.
  • Il permet de linéariser des produits : ln⁡(ab)=ln⁡(a)+ln⁡(b)\ln(ab) = \ln(a) + \ln(b)ln(ab)=ln(a)+ln(b)﻿
  • Sa réciproque, appelée exponentielle, est notée exp⁡(x)\exp(x)exp(x)﻿ ou exe^xex﻿.
  • Ensemble, elles permettent de généraliser la notion de puissance à tout réel : xα=exp⁡(α⋅ln⁡(x))pour x>0x^\alpha = \exp(\alpha \cdot \ln(x)) \quad \text{pour } x>0xα=exp(α⋅ln(x))pour x>0﻿
Chapitre 5.2 — Le logarithme

1. 🎯 Objectif

Ce chapitre définit rigoureusement la fonction logarithme naturel ln⁡(x)\ln(x)ln(x)﻿, en étudie les propriétés fondamentales, ses opérations algébriques et ses graphes, ainsi que ses liens avec les puissances et l'exponentielle.

2. 📌 Définition rigoureuse

🔹 Logarithme naturel

Pour tout réel x>0x > 0x>0﻿, on définit :

ln⁡(x):=∫1x1t dt\boxed{\ln(x) := \int_1^x \frac{1}{t} \,dt}ln(x):=∫1x​t1​dt​﻿

  • Si x>1x > 1x>1﻿, l'aire est positive
  • Si x=1x = 1x=1﻿, l'aire est nulle
  • Si x<1x < 1x<1﻿, l'aire est négative

👉 ln⁡(x)\ln(x)ln(x)﻿ est donc bien définie pour tout x>0x > 0x>0﻿.

Chapitre 5.3 — Fonction exponentielle

1. 🎯 Objectif

Comprendre la fonction exponentielle exp⁡(x)\exp(x)exp(x)﻿ (ou exe^xex﻿) comme la réciproque du logarithme naturel, en étudier les propriétés, valeurs, dérivées, et comportements asymptotiques.

2. 📌 Définition rigoureuse

La fonction exponentielle est définie comme l'inverse de la fonction logarithme :

exp⁡(x)=l’unique y>0 tel que ln⁡(y)=xpour tout x∈R\boxed{ \exp(x) = \text{l'unique } y > 0 \text{ tel que } \ln(y) = x } \quad \text{pour tout } x \in \mathbb{R}exp(x)=l’unique y>0 tel que ln(y)=x​pour tout x∈R﻿

On note aussi :

exp⁡(x)=ex\exp(x) = e^xexp(x)=ex﻿

3. 🔄 Relation fondamentale

ln⁡(ex)=xeteln⁡(x)=xpour x>0\boxed{ \ln(e^x) = x \quad \text{et} \quad e^{\ln(x)} = x \quad \text{pour } x > 0 }ln(ex)=xeteln(x)=xpour x>0​﻿

Ces identités expriment le fait que exp⁡\expexp﻿ et ln⁡\lnln﻿ sont réciproques l'une de l'autre.

Chapitre 5.4 — Base quelconque

1. 🎯 Objectif

Ce chapitre permet :

  • de généraliser les fonctions exponentielle et logarithme à une base a≠ea\ne ea=e﻿,
  • de comprendre les liens entre logarithmes de différentes bases,
  • de manipuler puissances, produits et dérivées avec ces nouvelles fonctions.

2. 📌 Définition — base a>0a > 0a>0﻿, a≠1a \ne 1a=1﻿

🔹 Exponentielle en base aaa﻿

ax:=ex⋅ln⁡(a)pour tout x∈R\boxed{ a^x := e^{x \cdot \ln(a)} \quad \text{pour tout } x \in \mathbb{R} }ax:=ex⋅ln(a)pour tout x∈R​﻿

Cette définition est compatible avec les puissances entières, rationnelles, et réelles.
